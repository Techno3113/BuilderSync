# Support

BuilderSync is completely free and open source — no ads, no paywalls, no accounts required.

## Found a bug?

Open an issue on GitHub:
**[github.com/Techno3113/BuilderSync/issues](https://github.com/Techno3113/BuilderSync/issues)**

Please include:
- What you were doing when it happened
- Your GD version and platform (Mac/Windows)
- The Geode log output if available (`geode/log/` folder)

## Want to contribute?

Pull requests are welcome! Check the README for build instructions.

Key areas that need help:
- Full late-join level sync
- Trigger and object property syncing
- Undo stack isolation for remote ops
- Android build testing

## Show your support

- ⭐ Star the repo on GitHub
- Share BuilderSync with other GD creators
- Submit your collaboratively-built levels to the GD servers and mention BuilderSync in the description!
