# Changelog

## v1.0.0
- Initial release
- Real-time object placement, deletion, move, and rotation sync
- Ghost cursors with per-player colour and name tag
- Host/Join popup with room code system
- User list popup (accessible from editor pause menu)
- Join/leave toast notifications
- Send queue — ops sent before WebSocket opens are buffered and flushed automatically
- Relay server (`server/server.js`) with room management and host promotion on disconnect
- Settings: relay server URL, display name, cursor sync toggle, notification toggle
